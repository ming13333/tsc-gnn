## 3. 逻辑（Logic）— 论证链与内部一致性

### 🔴 致命：中心前提与底层数据直接矛盾（Repro-Guard 已确认）

稿件在 **6 处**反复断言"图在 0/90 配置优于线性"：
- Highlights #1：「A fixed causal graph yields no prediction gain over linear baselines.」
- eTOC：「graph topology does not improve perturbation prediction beyond linear baselines (0/90)」
- Summary：「graph topology does not improve perturbation prediction beyond linear baselines (0/90)」
- §3.1：「it beat the linear baseline in 0 cases and showed no significant difference in 86 cases」
- §4.1：「the graph beat the linear baseline in zero cases」
- Conclusion：「does not beat linear models at prediction」

但 `gate1/gears_validation/robustness_results.csv` 的 `sig` 列实测为：
- **24 `better` / 17 `worse` / 174 `n.s.`**
- **combo（组合扰动）任务**上 base-graph rel_imp ≈ **+18%~+19%**，且多数种子 CI 不含 0 → 图**显著更优**。

**影响**：
1. 整篇"图无预测优势 → 只能卖可解释性"的主轴，建立在数据不支持的断言上。
2. §3.1→§3.2 的论证前提（"既然图不预测更好，为何还用图"）是假的；不过 §3.2 的"边级输出只能由图产生、线性模型无法产出"这一论点**本身独立成立**，可保留。
3. 此前该错误已被静默写进 MEMORY.md 并贯穿多轮（典型 AI silent error）。已更正记忆，结论真伪待你拍板。

**必须决策（二选一）**：
- **A. 改数字**：combo 任务上图显著更优、single 任务上持平/略差；重述 §3.1、Summary、eTOC、Highlights#1、§4.1、Conclusion。
- **B. 改主轴定位**：既然图在组合扰动上确实更好，反而支持图方法，主轴从"可解释性唯一卖点"调整为"组合扰动预测 + 边级可解释性双卖点"。

### 🟠 论证链其余检查
- §3.1 single 任务：ridge 持平略差、kernel 过拟合（rel≈−126%）→ "固定图+线性读头在 single 上无增益"成立，但**不应外推到 combo**。
- §4.3「边不稳定是生物学发现而非局限」：论证强，引用 degeneracy / redundancy / attractor 网络生物学原理 → 优秀。
- L5 三层三角验证 + A+C 综合：诚实、含 K562 正对照，合规且强。
- 时间轴拼接（24h→2d→14d 跨队列）已在 §2.1 / §4.3 / §4.4 多处诚实披露 → 好。
