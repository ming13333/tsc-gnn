## 一、Patterns 格式规范要点（实测自官网）

| 项目 | Patterns 要求 |
|---|---|
| **Summary（摘要）** | 单段落，**≤150 词**；含背景/目的、主要结果、方法/模型、结论、更广意义；**不引文献**；避免 "novel"/"first" |
| **Keywords** | **≤10 个**，逗号分隔 |
| **正文引用** | **顺序上标编号**（如 `Smith et al.¹` 或 `observation1,2`），**不是 [1] 方括号** |
| **参考文献列表** | 顺序编号；格式：`Author, A., and Author, B. (Year). Title. Journal V, P–P. DOI.`（期刊名不缩写过度、卷斜体、DOI 置末） |
| **Bigger picture 框** | **所有 Patterns 文章必需**；1–2 段，**≤300 词**；非摘要重述；面向非专家，可含前瞻性/推测 |
| **Resource availability** | **所有研究文章必需**；含三个子节：**Lead contact / Materials availability / Data and code availability** |
| **Highlights（终稿）** | ≤4 条，每条 **≤85 字符**，避免缩写 |
| **eTOC blurb（终稿）** | **≤50 词**，2 句，第三人称，面向大众 |
| **Graphical abstract** | Patterns 在"可选"期刊列表内（Cell Systems / Current Biology / Patterns / Neuron / Newton）——**非必需** |
| **结构** | 通常为 introduction→results→discussion→methods→references；首投允许变通排序 |
| **主文字数** | 研究文章典型 **5,000–7,000 词**；无硬性上限（"length aligns with science" 时可更长） |
| **图/表数量** | 研究文章**无上限** |
| **终稿文件** | 主文（标题至参考文献+图注）单 Word 文档；图单独上传；Declaration of interests 表单需随稿 |

## 三、优先修改清单与精简策略

### A. 投稿前必须改（blocking，约 1–2 天可完成）
1. **Summary 重写**为 ≤150 词单段。

3. **参考文献列表**全量重排为 Cell Press 式（剥离编辑注、改 & 为 ", and"、年提前、去加粗、补 DOI）。
4. **新增 Bigger picture 框**（≤300 词）。
5. **Resource availability** 改为 Lead contact / Materials availability / Data and code availability 三子节。
6. **Keywords** 删至 ≤10。

### B. 强烈建议（提升录用概率）
7. **主文精简至 ~7,000–8,000 词**：
   - 将 Table 2、3、4、6、7、8、9、10、11 中至少 6 个移入 Supplementary（保留 Table 1 证据阶梯 + 1–2 个核心表于正文）；
   - 压缩 §4.3 "Edge instability as a biological finding"（目前最长，且与 §4.2 有重叠）与 §4.4 Limitations；
   - 合并 §4.1–4.2 的部分重复论证。
8. **章节标题**去 `§` 符号，改描述性粗体（Methods 论文可接受保留层级，但去掉 § 记号）。

### C. 终稿阶段（accept 后）

## 四、可直接套用的草稿（示例）

### 4.1 重写后的 Summary（≈145 词，单段，无文献）
> Virtual perturbation methods for temporal gene-regulatory-network (GRN) rewiring either lack temporal resolution, ignore cell–cell communication, or sacrifice interpretability for accuracy. We present TSC-GNN, an interpretable framework that recovers edge-level GRN rewiring, master-regulator dynamics, and drug-repurposing hypotheses from multi-timepoint single-cell transcriptomes. In ischemic stroke (two mouse cohorts, 24 h–14 d), it recovers the canonical repair program (Sox10→myelin, ΔW = +0.51) with cross-cohort reproducibility (ρ = 0.48–0.55) and human conservation (SOX10→myelin OR = 27). Triangulating three public perturbation modalities shows program-level causal support is context-gated. Under a fixed causal graph with a linear readout, graph topology does not improve perturbation prediction beyond linear baselines (0/90); the contribution is interpretability, not accuracy.

### 4.2 Bigger picture 框草稿（≈210 词，非专家可读）
> Most computational tools that predict how cells respond to perturbations are judged by a single metric: prediction accuracy. As models grow larger and more opaque, they increasingly hide the very mechanism researchers want to understand — which regulatory connections change, and why. This paper takes the opposite stance: it treats the gene-regulatory network as a structured map on which a perturbation is decomposed into individual edge-level changes, and it openly reports that this map does not make predictions more accurate than a simple linear model. What it provides instead is interpretability: a transparent, testable account of which transcription factors rewire across a biological transition.
>
> The implications reach beyond stroke. Any field using single-cell transcriptomics to study temporal remodeling — development, neurodegeneration, cancer, immunology — faces the same trade-off between black-box accuracy and mechanistic clarity. More broadly, the finding that individual regulatory edges are context-specific while whole programs are robust argues for a field-wide recalibration: edge-level output should be presented as hypotheses to test, not settled causal truth. As single-cell perturbation datasets multiply, frameworks that pair interpretability with honest boundary-setting will be essential for turning that data into trustworthy biology.

### 4.3 参考文献格式对照（必须全量重排）
- **当前（错误）**：`[1] Cramer, S. C. & Carrico, C. R. Stroke recovery: a perspective on mechanisms. *Nat. Rev. Neurosci.* **9**, 720–731 (2008). — Stroke recovery mechanisms overview.`
- **Patterns 要求（正确）**：`Cramer, S.C., and Carrico, C.R. (2008). Stroke recovery: a perspective on mechanisms. Nat. Rev. Neurosci. 9, 720–731. 10.1038/nrn2474.`
- 改动点：去 `[1]` → 顺序号；`&` → `, and`；年提前至作者后 `(2008).`；去 `*Journal*`/`**V**` 加粗（卷可斜体）；删编辑注 `— ...`；补 DOI。
- 注：当前 56 条文献每条都带"—— 注释"，**重排时需逐条剥离**；建议用脚本先批量去注释与去加粗，再人工补 DOI 与期刊标准缩写。

### 4.4 Highlights 草稿（≤85 字符/条，无缩写）
- A fixed causal graph yields no prediction gain over linear baselines.
- Edge-level rewiring is recovered as interpretable, testable hypotheses.
- Evidence ladder escalates from reproducibility to causal support.
- Program-level causal support is context-gated across perturbations.

### 4.5 eTOC blurb 草稿（≤50 词，第三人称）
> This framework recovers interpretable edge-level rewiring of gene networks from single-cell data. It shows that a fixed causal graph aids interpretation, not prediction accuracy, and that regulatory programs—not individual edges—are robust across contexts. (48 words)

## 五、工作量评估与可执行性

| 任务 | 可脚本化？ | 说明 |
|---|---|---|
| 引用 [N]→上标 | ✅ 高 | 正则批量替换，需校核多引用 `[1, 2]`→`1,2` 上标 |
| 参考文献重排 | ⚠️ 部分 | 去注释/去加粗可脚本；**DOI 与期刊标准缩写需逐条人工补** |

| Resource availability 三子节 | ✅ 高 | 直接改标题层级 |
| 主文精简/移表 | ❌ 人工 | 需判断，我可给具体删减段落建议 |
| 标题去 § 符号 | ✅ 高 | 批量替换 |

**整体判断**：格式层面有 5 项 blocking 改动，其中引用与文献重排可脚本化约 70%，其余为撰写型。预计一轮修改（脚本 + 人工撰写）可在 1–2 天内完成到"可投 Patterns"状态。
