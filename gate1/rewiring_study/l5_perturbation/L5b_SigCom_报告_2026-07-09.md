# L5b · SigCom LINCS 基因扰动方向一致性分析报告

**日期**: 2026-07-09  
**定位**: L5 causal support 的补充层——独立基因扰动库中 TF 自身扰动签名的方向一致性检验

---

## 1. 设计逻辑

| 层面 | 设计 |
|------|------|
| **查询输入** | Human DoRothEA 靶程序（SOX10/CEBPB/GATA2 靶基因集 → entity IDs） |
| **查询数据库** | SigCom LINCS: `l1000_xpr`（CRISPR KO, ~14 万签名）+ `l1000_oe`（过表达, ~3.4 万签名） |
| **预期方向** | CRISPR KO 激活因子 → 靶基因下调 → 出现为 **reverser**；OE 激活因子 → 靶基因上调 → 出现为 **mimicker** |
| **自 vs 交叉特异性** | 3×3 矩阵：TF_query 靶程序 → TF_own 扰动签名排名；对角线（self）应优于非对角线（cross） |
| **靶程序来源** | 独立 DoRothEA GRN（非循环论证） |
| **红线** | 只验 TF/模块级方向一致性，不逐边验证；定位 causal support 非 validation |

## 2. 技术细节

- **API**: SigCom LINCS `data-api/api/v1/enrich/ranktwosided`，`database` 参数用字符串名（`l1000_xpr`/`l1000_oe`），非 UUID
- **靶集大小**: SOX10=321 entities, CEBPB=500 (capped from 587), GATA2=500 (capped from 5262)
- **down_entities**: 每 TF 独立选取 5 个中性基因（排除该 TF 靶集），满足 API ≥5 要求
- **limit=2000**: 返回 top 2000 mimickers + top 2000 reversers = 4000 签名/查询
- **排名解读**: rank 0 = z-sum 最高 = 最强 mimicker；rank maxRank = z-sum 最低 = 最强 reverser

## 3. 结果

### 3.1 CRISPR Knockout (l1000_xpr, maxRank=140,603)

| Query → Own | Rank | Type | Percentile | p-value | Self? |
|---|---|---|---|---|---|
| SOX10 → SOX10 | 139,333 | reverser | top 0.90% | 7.4e-02 | **SELF** |
| SOX10 → CEBPB | 139,116 | reverser | top 1.06% | 4.5e-04 | cross |
| SOX10 → GATA2 | 139,217 | reverser | top 0.99% | 1.7e-03 | cross |
| CEBPB → CEBPB | 139,956 | reverser | top 0.46% | 6.1e-02 | **SELF** |
| CEBPB → CEBPB | 139,667 | reverser | top 0.67% | 9.2e-04 | **SELF** |
| CEBPB → CEBPB | 570 | mimicker | top 0.41% | 3.1e-08 | **SELF** (wrong dir) |
| GATA2 → GATA2 | 138,952 | reverser | top 1.17% | 3.9e-02 | **SELF** |
| GATA2 → SOX10 | 139,136 | reverser | top 1.04% | 3.7e-01 | cross |
| GATA2 → CEBPB | 1,215 | mimicker | top 0.86% | 2.8e-03 | cross |

**自特异性**: 
- SOX10: self=0.90% vs best_cross=1.06% → delta=-0.15%（self 略弱于 cross，**无自特异性**）
- GATA2: self=1.17% vs best_cross=1.04% → delta=+0.13%（self 略弱于 cross，**无自特异性**）
- CEBPB: cross 全部 absent，**无法评估**

**方向一致性**: 全部 3 个 TF 的 CRISPR KO 均以 reverser 方向出现（✓ 正确），且均在 top ~1% 最强 reverser 中。

### 3.2 Overexpression (l1000_oe, maxRank=33,782)

| Query → Own | Rank | Type | Percentile | p-value | Self? |
|---|---|---|---|---|---|
| **GATA2 → GATA2** | **3** | **mimicker** | **top 0.01%** | **1.4e-05** | **SELF** |
| GATA2 → GATA2 | 1,055 | mimicker | top 3.12% | 2.2e-02 | **SELF** |
| SOX10 → SOX10 | 32,109 | reverser | top 4.95% | 3.8e-02 | **SELF** (wrong dir) |
| SOX10 → SOX10 | 1,261 | mimicker | top 3.73% | 1.8e-03 | **SELF** |
| GATA2 → SOX10 | 32,101 | reverser | top 4.98% | 1.1e-01 | cross |
| CEBPB → GATA2 | 1,139 | mimicker | top 3.37% | 1.9e-04 | cross |

**自特异性**:
- GATA2: self=0.01% vs best_cross=4.98% → delta=-4.97%（**强自特异性！** GATA2 OE 是自身靶程序的第 3 强 mimicker，而交叉 TF 在 ~5%）
- SOX10: self=4.95% (reverser, wrong direction) vs best_cross=2.87% → **方向错误且无自特异性**
- CEBPB: OE 库中无 CEBPB 扰动签名，**无法评估**

## 4. 核心发现

### ✅ GATA2 过表达 — 金标准结果

GATA2 OE 签名在 GATA2 靶程序查询中排名 **第 3/33,782**（top 0.01%），type=mimicker（方向正确：OE 激活因子 → 靶基因上调 → mimics up=target_set），p=1.4e-05，且 **2/2 自身签名均为 mimicker**。自特异性极强：self=0.01% vs cross=4.98%，delta=-4.97%。

这是 L5b 最强证据：**独立 LINCS 过表达数据正交验证了 GATA2 作为激活因子的方向性因果预测**。

### ~ CRISPR Knockout — 方向正确但无自特异性

全部 3 个 TF 的 CRISPR KO 均以 reverser 出现（方向正确），均在 top ~1%。但 self 与 cross 排名接近（delta ±0.15%），**不具备 TF 级自特异性**。可能原因：
1. L1000 CRISPR KO 在细胞系中进行，TF 作用谱可能重叠
2. 仅 5 个中性 down 基因提供的 gene-set 特异性有限
3. 三个 TF 共享部分下游通路（炎症/髓鞘）

### ✗ SOX10 过表达 — 方向不一致

SOX10 OE 同时出现为 reverser (rank 32109) 和 mimicker (rank 1261)，主导方向为 reverser（错误）。可能在 L1000 细胞系中 SOX10 OE 未有效激活其神经特异性靶程序。

### — CEBPB 过表达 — 无数据

L1000 OE 库中无 CEBPB 扰动签名。

## 5. 证据等级定位

| 发现 | 等级 | 定位 |
|------|------|------|
| GATA2 OE rank 3 mimicker, self-specific | **强支持** | 独立扰动数据正交验证 GATA2 激活因子方向性 |
| 3 TFs CRISPR KO reverser direction | **中等支持** | 方向一致但缺自特异性 |
| SOX10 OE wrong direction | **不支持** | 诚实披露 |
| CEBPB OE no data | **N/A** | 数据缺失 |

**整体定位**: L5b 提供了 **部分方向性因果支持**——GATA2 在独立 OE 库中获得强自特异性验证，CRISPR KO 方向一致但特异性不足。不可声称全部 TF 均获验证。

## 6. 诚实边界

1. **CAP 截断**: CEBPB/GATA2 靶集分别从 587/5262 截断至 500，可能丢失信息
2. **中性基因**: down_entities 仅 5 个中性基因，gene-set 特异性有限；CRISPR KO 无自特异性可能部分源于此
3. **细胞系**: L1000 在癌细胞系中进行，SOX10 神经特异性靶程序可能未有效激活
4. **CEBPB OE 缺失**: 无法评估 CEBPB 过表达方向
5. **SOX10 OE 方向错误**: 须诚实披露，不选择性报告
6. **单库验证**: 仅 LINCS L1000 一个数据源，非多平台共识
7. **定位**: causal support / directional consistency，**非 validation**

## 7. 与 L5 整合

L5（公开 KO 数据程序级方向性因果支持）已验证 Sox10/Cebpb 程序级方向一致性。L5b 进一步在独立 LINCS 基因扰动库中检验：

- **L5**: Sox10 cKO → 靶程序 rank 46/412 前 11%，OR=1.81；Cebpb 杂合 KO → rank 149/404 前 37%，OR=1.20
- **L5b**: GATA2 OE → 自身靶程序 rank 3/33782 top 0.01%（强自特异性）；3 TFs CRISPR KO 方向正确但无自特异性

两层互补：L5 验证鼠源 KO 数据程序级方向，L5b 在独立人源 LINCS 库中验证 GATA2 扰动方向。GATA2 是两层共同支持最强的 TF（L5 虽未直接验证 Gata2 KO，但 L5b OE 提供了强正交证据）。
